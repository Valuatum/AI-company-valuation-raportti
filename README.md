# Athlos Oy AI-arvonmaaritysraportti

Tama repo sisaltaa Athlos Oy:n paivitetyn AI-arvonmaaritysraportin standalone-HTML- ja PDF-versiot.

## Tiedostot

- `index.html` - paivitetty standalone-HTML, avattavissa suoraan selaimessa.
- `outputs/athlos-ai-arvonmaaritysraportti.html` - sama HTML toimituskopiona.
- `outputs/athlos-ai-arvonmaaritysraportti.pdf` - paivitetty PDF-versio.
- `scripts/` - paivitykseen ja tarkistuksiin kaytetyt Node-skriptit.

## Paivityksen sisalto

- Osioon 2 lisattiin julkiset verkkolahteet liiketoimintaprofiilia varten.
- Osio 3 korvattiin uudella liiketoimintaprofiililla, tuotekorteilla, arvonmaaritys-calloutilla ja lahderekisterilla.
- Sisallysluettelo ja sivunumerointi paivitettiin 22-sivuiseksi raportiksi.
