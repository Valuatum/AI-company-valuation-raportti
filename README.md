# Valuatum Oy AI-arvonmaaritysraportti

Tama repo sisaltaa Valuatum Oy:n AI-arvonmaaritysraportin standalone-HTML- ja PDF-versiot.

## Tiedostot

- `index.html` - Valuatum-raportin standalone-HTML, avattavissa suoraan selaimessa.
- `outputs/valuatum-ai-arvonmaaritysraportti.html` - sama HTML toimituskopiona.
- `outputs/valuatum-ai-arvonmaaritysraportti.pdf` - PDF-versio.
- `scripts/` - raportin rakentamiseen ja tarkistuksiin kaytetyt Node-skriptit.

## Paivityksen sisalto

- Liitetyn Valuatum-tekstin sisalto taitettiin samaan raporttipohjaan.
- Sisallysluettelo ja sivunumerointi muodostuvat renderoinnissa automaattisesti.
- Lopullinen PDF on tarkistettu 22-sivuiseksi ilman sivuylivuotoja.
